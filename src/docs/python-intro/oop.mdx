# Ojected Oriented Programming in Python

## What Is Object-Oriented Programming (OOP)?
Object-oriented Programming, or OOP for short, is a programming paradigm which provides
a means of structuring programs so that properties and behaviors are bundled into individual objects.

For instance, an object could represent a person with a name property, age, address, etc., with behaviors like walking, talking, breathing, and running. 
Or an email with properties like recipient list, subject, body, etc., and behaviors like adding attachments and sending.

Read more [here](https://realpython.com/python3-object-oriented-programming/).
