# Hello Word in Python


> A "Hello, World!" program generally is a computer program that outputs or displays the message "Hello, World!". 
> Such a program is very simple in most programming languages, and is often used to illustrate the basic syntax of a programming language.
> [Wikipedia](https://en.wikipedia.org/wiki/%22Hello,_World!%22_program)

Hello World in Python is: `print("Hello World")`

![Hello world en Python](./hello-world.png)


What exactly is going on?

`print()` shows the ouput. It's a function.

`"Hello World"` **is** the output. It's a string.

`"Hello World"` is also a parameter of the function `print()`
