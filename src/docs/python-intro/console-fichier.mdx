Pour cet atelier, on va coder en ligne. Normalement, on utiliserait un environnement de développement comme PyCharm ou Spyder. Les explications dessous sont généralement les mêmes. 


Il existe deux parties dessus : 

**Le fichier**

Le fichier est un endroit ou on code la plupart du temps

- Quand on 'lance' le fichier, on lance toutes les lignes de haut en bas
- La sortie n'est visible que avec `print`
- Le fichier est pour du code qui va être lancer plusieurs fois

**La console**

La console affiche les sorties de votre application (le code dans votre fichier). Vous pouvez aussi coder directement dans la console !

La console :

- lance chaque ligne une fois elle est écrite
- montre la sortie pour chaque ligne
- Est interactive et utilisée pour exploration


### Calculs

On peut taper directement dans la console :

- `2 + 3`
- `2 - 4` 

En programmation, le signe de division est `/` et le signe de multiplication est `*`. Donc :

- 2 `*` 4 = 8
- 10 `/` 2 = 5
