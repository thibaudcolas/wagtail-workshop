> La programmation informatique est l'écriture de programmes servant à créer un logiciel. Un programme contient l'ensemble des informations nécessaires pour que le logiciel remplisse sa tâche.
> [L'Internaute](https://www.linternaute.fr/dictionnaire/fr/definition/programmation-informatique/)

**La programmation est la création des logiciels.**

### Il y a plusieurs langages de programmation

En général, certains langages créent certaines applications. Par exemple, le JavaScript est utilisé pour les sites web, Swift pour les applications iPhone et SQL pour les bases de données. C'est pas possible d'utiliser Swift pour les bases de données. Il existe pleins de langages utilisés pour une pléthore de choses comme Java, Python et C.

Normalement, la programmation a lieu dans un environnement de développement, un logiciel pour créer les logiciels (méta).
