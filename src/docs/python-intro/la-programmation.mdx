### There are many programming languages

In general, certain programming languages are used for certain applications. For example, JavaScript is used for websites, Swift for iPhone apps and SQL for databases.
There are some languages that are used for a wide range of things like Java, **Python** and C.

Normally, programming takes place in an Integrated Development Environment.
